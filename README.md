# Clínica Reinventate - Centro de Estética

Sitio web premium para clínica de estética y belleza en San Miguel de Tucumán.

## Tecnologías

- HTML5
- CSS3 (Sin frameworks)
- JavaScript Vanilla

## Características

- Diseño responsive
- Navbar con menú hamburguesa en móvil
- Sección de servicios con tarjetas interactivas
- Collage de imágenes con efecto blanco y negro en la sección de filosofía
- Testimonios con fotos y hover animations
- Formulario de contacto con labels flotantes y validación
- Footer centrado con logo e iconos de redes sociales
